# BrightCart - JavaScript Framwork Assigment 

### Live demo

Link to demo site - [BrightCart](effortless-gumdrop-3a143f.netlify.app)

## Description
BrightCart is a simple yet polished e-commerce application built with React, designed to showcase the core concepts and skills gained through working with JavaScript frameworks. The application features a clean and intuitive user interface for searching products, managing a shopping cart, and completing purchases.

## Technologies Used

- React
- React Router
- Tailwind Css
- Vite
- Prettier
- ESLint

## Getting Started

- Make sure you have Node.js installed
- I you don't have Node.js, downloade it her --> [Node.js](https://nodejs.org/en)

**Clone the repository**:

1. Clone the repository

```bash
git clone https://github.com/Iseost/online-store-assigment.git
```

Or: open with Githib Desktop

2. Navigate to the project directory

```bash
cd vite-project
```

3. Install despendencies:
```bash
npm install
```

5. Run the devolopment: 
```bash
npm run dev
```
